Silkroad-Orginal runtime folder

This is the only runtime profile used by PK2 Tools.
GFXFileManager.dll is copied here during build.

Default PK2 Blowfish Key: 169841
The GUI field PK2 BLOWFISH KEY can override this value for read/build/import/extract operations.


## GUI Workspaces

1. Encryptor: encrypt/decrypt loose folders or internal PK2 payloads.
2. Extractor: read and extract PK2 contents using the selected Blowfish key.
3. Import / Inject: import files or folders into an existing PK2.
4. Build PK2: build PK2 archives from source folders using the selected Blowfish key.

Default PK2 Blowfish Key: 169841. Change it in the GUI before reading or building to use another key.
